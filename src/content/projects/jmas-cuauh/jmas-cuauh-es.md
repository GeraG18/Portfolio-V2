# JMAS - Cuauhtémoc

**JMAS Gestión del Agua** es un sistema especializado desarrollado por Córdova Industrial Integradores (CIDTcor) para la Junta Municipal de Agua y Saneamiento de Cuauhtémoc, diseñado para el control y monitoreo centralizado de pozos de agua en diversas zonas de la ciudad. Esta aplicación de acceso restringido permite a los operadores de JMAS gestionar de manera más eficiente y efectiva los recursos hídricos, representando una renovación completa de la solución anterior para abordar problemas críticos de rendimiento, codificación y funcionalidad.

## Características principales

- Control centralizado de pozos de agua municipales
- Monitoreo en tiempo real de niveles y calidad del agua
- Sistema de notificaciones personalizadas basadas en condiciones específicas
- Interfaz optimizada para gestión operativa eficiente
- Acceso restringido exclusivo para personal autorizado de JMAS

### Responsabilidades y Logros Clave:

- **Contribuí al rediseño completo de la aplicación** identificando y proponiendo soluciones para problemas críticos de rendimiento y codificación del sistema anterior
- **Diseñé y desarrollé la estructura de UX/UI** creando interfaces intuitivas para la gestión de recursos hídricos
- **Estructuré y optimicé componentes de la base de datos** para el manejo eficiente de datos de monitoreo de pozos
- **Implementé lógica avanzada en Node-RED** desarrollando sistemas para obtención y subida de datos, además de funciones de notificación personalizadas basadas en condiciones específicas
