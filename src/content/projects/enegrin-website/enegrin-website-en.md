# Enegrin Website

**Enegrin Website** is the corporate website developed for Córdova Industrial Integradores' (CIDTcor) Enegrin project, created to present details of this innovative green energy platform designed for Mexican production companies. The website displays comprehensive information about Enegrin's objectives, vision, mission, and applications, while highlighting its benefits in terms of energy quality, stability, and operational continuity for businesses. Developed in WordPress, the site combines business functionality with attractive design.

## Main Features

- Complete corporate presentation of the Enegrin project
- Detailed information about vision, mission, and objectives
- Applications and use cases section
- Responsive design optimized for multiple devices
- Integration with WordPress tools

### Key Responsibilities and Achievements:

- **Designed and developed the UX/UI structure** creating an intuitive and attractive experience for visitors
- **Optimized website performance** implementing improvements in loading speed and efficiency
- **Developed custom CSS** to enhance and customize native WordPress components
- **Implemented security improvements** strengthening site protection against common vulnerabilities
