# Página Web Enegrin

**Página Web Enegrin** es el sitio web corporativo desarrollado para el proyecto Enegrin de Córdova Industrial Integradores, creado para presentar los detalles de esta innovadora plataforma de energía verde diseñada para empresas productoras mexicanas. El sitio web muestra información completa sobre los objetivos, visión, misión y aplicaciones de Enegrin, además de destacar sus beneficios en términos de calidad, estabilidad energética y continuidad operativa para las empresas. Desarrollado en WordPress, el sitio combina funcionalidad empresarial con un diseño atractivo.

## Características principales

- Presentación corporativa completa del proyecto Enegrin
- Información detallada sobre visión, misión y objetivos
- Sección de aplicaciones y casos de uso
- Diseño responsivo optimizado para múltiples dispositivos
- Integración con herramientas de WordPress

### Responsabilidades y Logros Clave:

- **Diseñé y desarrollé la estructura de UX/UI** creando una experiencia intuitiva y atractiva para los visitantes
- **Optimicé el rendimiento del sitio web** implementando mejoras en velocidad de carga y eficiencia
- **Desarrollé CSS personalizado** para mejorar y personalizar los componentes nativos de WordPress
- **Implementé mejoras de seguridad** fortaleciendo la protección del sitio contra vulnerabilidades comunes
