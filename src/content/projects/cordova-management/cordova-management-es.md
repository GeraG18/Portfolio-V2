# Aplicación Web Obras Córdova

**Obras Córdova** es un proyecto ERP para la gestión de los proyectos realizados por parte de Córdova Industrial Integradores a empresas de producción industrial que requieran una automatización, esta aplicación brinda la posibilidad de calcular costes de uso de materiales, controlar y gestionar el horario de los trabajadores y el bono que recibirán por la realización de ciertas tareas.

## Características principales

- Gestión de horarios de trabajo de cada trabajador.
- Control de los bonos por trabajador en base a las tareas realizadas durante la jornada laboral.
- Gestión de costes de uso de materiales.
- Aprobación de ordenes de compra y adjuntación de archivos como la factura y otros elementos requeridos.
- Control del coste total del proyecto el cual se está realizando.
- Inventario de materia prima y herramientas que se usan en cada proyecto.

### Responsabilidades y Logros Clave:

- **Lideré el diseño de UX/UI** completo, creando interfaces intuitivas y accesibles para todos los usuarios utilizando Figma.
- **Desarrollé el prototipo de la interfaz funcional** implementando el diseño en Ionic con Angular y Tailwind CSS, estableciendo las bases técnicas del producto e implementando el diseño responsivo.
- **Realicé los algoritmos y calculos complejos para el reporte financiero** por proyecto y el reporte financiero de todos los proyectos.
- **Configuré y realicé varias partes de la base de datos en Firebase** para gestionar de forma optimizada los datos en la base de datos NoSQL.
- **Implementé el sistema de autenticación seguro** integrando un proveedor único (Email) con verificación de dominio.
- **Desarrollé la lógica para el control de inventario (almacén)** para realizar la salida de entrada de materiales y/o herramienta.
