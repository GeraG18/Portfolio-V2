# Obras Córdova Web Application

**Obras Córdova** is an ERP project for managing projects carried out by Córdova Industrial Integradores for industrial production companies requiring automation. This application provides the ability to calculate material usage costs, control and manage worker schedules, and track bonuses received for completing specific tasks.

## Main Features

- Management of work schedules for each worker
- Control of bonuses per worker based on tasks completed during the workday
- Management of material usage costs
- Approval of purchase orders and attachment of files such as invoices and other required elements
- Control of the total cost of the project being carried out
- Inventory of raw materials and tools used in each project

### Key Responsibilities and Achievements:

- **Led the complete UX/UI design**, creating intuitive and accessible interfaces for all users using Figma
- **Developed the functional interface prototype** by implementing the design in Ionic with Angular and Tailwind CSS, establishing the technical foundations of the product and implementing responsive design
- **Created algorithms and complex calculations for financial reporting** per project and overall project financial reporting
- **Configured and developed various parts of the Firebase database** to optimally manage data in the NoSQL database
- **Implemented the secure authentication system** integrating a single provider (Email) with domain verification
- **Developed the inventory control logic (warehouse)** to manage material and/or tool input and output
