# Dealer Kiosk System - Horizon Trailers

**Dealer Kiosk System** is a hardware/software solution developed for NATM and NATDA trade shows, designed to capture applications from dealers interested in becoming certified sellers for Horizon Trailers. The application runs on a Raspberry Pi with a touchscreen display and presents a form optimized for touch input, allowing show visitors to apply directly from the exhibition booth.

## Main Features

- Touch-optimized interface for kiosk displays
- Complete dealer application form with real-time validation
- Custom touch keyboard for data entry
- Local storage of applications on Raspberry Pi
- Automatic data backup system
- Administration interface for application review
- Integration with email notification system

### Key Responsibilities and Achievements:

- **Developed the complete application in Express.js** optimized for Raspberry Pi execution
- **Designed the touch interface in Figma** considering ergonomics and usability in trade show environments
- **Implemented custom touch keyboard** with specific validations for contact data
- **Configured and optimized Raspberry Pi** for continuous and stable operation in commercial settings
- **Created robust storage system** with automatic backups to prevent data loss
- **Developed administration interface** for sales team to review captured applications
- **Implemented email system integration** for automatic notifications to prospects
