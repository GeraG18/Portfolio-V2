# Sistema Kiosk para Dealers - Horizon Trailers

**Sistema Kiosk para Dealers** es una solución hardware/software desarrollada para ferias comerciales NATM y NATDA, diseñada para capturar solicitudes de dealers interesados en convertirse en vendedores certificados de Horizon Trailers. La aplicación se ejecuta en una Raspberry Pi con pantalla táctil y presenta un formulario optimizado para entrada táctil, permitiendo a los visitantes de las ferias aplicar directamente desde el stand.

## Características principales

- Interfaz táctil optimizada para pantallas de kiosk
- Formulario completo de solicitud para dealers con validación en tiempo real
- Teclado táctil personalizado para entrada de datos
- Almacenamiento local de solicitudes en Raspberry Pi
- Sistema de respaldo automático de datos
- Interfaz de administración para revisión de solicitudes
- Integración con sistema de notificaciones por email

### Responsabilidades y Logros Clave:

- **Desarrollé la aplicación completa en Express.js** optimizada para ejecución en Raspberry Pi
- **Diseñé la interfaz táctil en Figma** considerando ergonomía y usabilidad en entorno de feria
- **Implementé el teclado táctil personalizado** con validaciones específicas para datos de contacto
- **Configuré y optimicé la Raspberry Pi** para ejecución continua y estable en entorno comercial
- **Creé sistema de almacenamiento robusto** con respaldos automáticos para prevenir pérdida de datos
- **Desarrollé interfaz de administración** para que el equipo de ventas revise solicitudes capturadas
- **Implementé integración con sistema de email** para notificaciones automáticas a prospectos
