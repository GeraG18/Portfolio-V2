# Nueva Aplicación de InteliHogar

**InteliHogar** es un proyecto innovador desarrollado por el área de Innovación y Desarrollo Tecnológico de Córdova Industrial (CIDTcor), diseñado para proporcionar un ecosistema completo de casa inteligente accesible con solo unos clics. La aplicación permite a los usuarios controlar y gestionar todos sus dispositivos inteligentes instalados en el hogar, ofreciendo funcionalidades avanzadas como organización por habitaciones, control total de dispositivos y sistema de invitaciones para compartir el control específico con otros usuarios.

## Características principales

- Control centralizado de dispositivos inteligentes del hogar
- Organización por habitaciones y zonas
- Sistema de invitaciones con permisos granularizados
- Gestión completa del ecosistema IoT
- Interfaz intuitiva para configuración rápida

### Responsabilidades y Logros Clave:

- **Diseñé y desarrollé la estructura de UX/UI** creando interfaces intuitivas para la gestión de dispositivos inteligentes
- **Estructuré y optimicé la base de datos** para el manejo eficiente de datos de dispositivos y configuraciones
- **Implementé el sistema de autenticación multifactor** integrando proveedores Google, Apple y Email con configuraciones específicas para iOS/iPadOS
- **Desarrollé la identidad visual completa** y branding utilizado en las tiendas de aplicaciones oficiales
- **Colaboré en la implementación de transmisión de datos** entre la aplicación y los dispositivos IoT del hogar
- **Optimicé y refactoricé código existente** mejorando significativamente el rendimiento de la aplicación
- **Participé en el desarrollo de un sistema de alarma inteligente** integrado con los dispositivos del hogar
- **Gestioné el proceso completo de publicación** en AppStore para plataformas iOS e iPadOS
