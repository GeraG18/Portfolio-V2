# Sistema Kiosk para Dealers - LoadBoss

**Sistema Kiosk para Dealers LoadBoss** es una adaptación especializada del sistema kiosk para capturar solicitudes de distribuidores especializados en plataformas para tractocamiones. Desarrollado para ferias comerciales del sector transporte pesado, esta versión incluye campos específicos para experiencia en equipos de carga, tipos de vehículos atendidos y capacidades de instalación, optimizando la captación de distribuidores calificados para productos LoadBoss.

## Características principales

- Formulario especializado para distribuidores de equipos pesados
- Campos específicos para experiencia en tractocamiones y plataformas
- Validación de capacidades técnicas y certificaciones
- Sistema de pre-calificación automática basado en experiencia
- Integración con base de datos de distribuidores existentes
- Reportes de métricas específicas para el sector transporte

### Responsabilidades y Logros Clave:

- **Adapté la plataforma kiosk existente** para requisitos específicos del sector transporte pesado
- **Diseñé formularios especializados** que capturan experiencia relevante en equipos LoadBoss
- **Implementé sistema de pre-calificación** que evalúa automáticamente la idoneidad de distribuidores
- **Integré con base de datos de dealers existente** para evitar duplicaciones y enriquecer datos
- **Desarrollé módulos de reporting específicos** para análisis de mercado de distribuidores
- **Optimicé interfaz para uso rápido** considerando el flujo de visitantes en ferias industriales
