# Cotizador Enegrin

**Cotizador Enegrin** es una herramienta especializada desarrollada como parte del proyecto Enegrin de Córdova Industrial Integradores, diseñada para calcular presupuestos detallados sobre el costo de instalación de sistemas de energía verde para empresas que requieren servicios energéticos de alta calidad. Esta plataforma de acceso restringido está integrada de manera discreta en el sitio principal del proyecto y requiere una clave de acceso única para que las empresas puedan visualizar sus cotizaciones personalizadas.

## Características principales

- Sistema de cálculo de presupuestos para instalaciones de energía verde
- Acceso restringido mediante claves de autorización únicas
- Interfaz privada integrada en la plataforma principal Enegrin
- Generación de cotizaciones personalizadas por empresa
- Almacenamiento seguro de datos de cotización

### Responsabilidades y Logros Clave:

- **Diseñé y desarrollé la interfaz de usuario completa** creando una experiencia intuitiva para el cálculo de cotizaciones
- **Estructuré y optimicé la base de datos** en Firestore para el almacenamiento seguro de datos de cotización
- **Implementé algoritmos de cálculo precisos** para la generación automática de presupuestos de instalación
