# Enegrin Quotation Tool

**Enegrin Quotation Tool** is a specialized tool developed as part of Córdova Industrial Integradores' (CIDTcor) Enegrin project, designed to calculate detailed budgets for green energy system installation costs for companies requiring high-quality energy services. This restricted-access platform is discreetly integrated into the project's main site and requires a unique access key for companies to view their personalized quotations.

## Main Features

- Budget calculation system for green energy installations
- Restricted access through unique authorization keys
- Private interface integrated into the main Enegrin platform
- Generation of customized quotations per company
- Secure storage of quotation data

### Key Responsibilities and Achievements:

- **Designed and developed the complete user interface** creating an intuitive experience for quotation calculations
- **Structured and optimized the database** in Firestore for secure storage of quotation data
- **Implemented precise calculation algorithms** for automatic generation of installation budgets
