# Sistema Kiosk para Distribuidores - Gearmax Pro

**Sistema Kiosk para Distribuidores Gearmax Pro** es la tercera implementación del sistema kiosk, desarrollada específicamente para la marca Gearmax Pro dentro del grupo H7X. Esta versión se enfoca en capturar distribuidores para equipamiento y accesorios de alto rendimiento, con formularios adaptados para evaluar experiencia en ventas de productos especializados, capacidad de inventario y alcance geográfico de distribución.

## Características principales

- Formulario optimizado para distribuidores de accesorios especializados
- Evaluación de capacidad de inventario y logística
- Mapeo de territorio de cobertura geográfica
- Sistema de scoring para priorización de distribuidores
- Integración con sistema de gestión de inventarios
- Dashboard en tiempo real de solicitudes durante ferias

### Responsabilidades y Logros Clave:

- **Desarrollé la tercera iteración del sistema kiosk** adaptada a productos Gearmax Pro
- **Implementé sistema de scoring avanzado** que prioriza distribuidores por potencial de ventas
- **Creé módulo de mapeo geográfico** para visualizar cobertura territorial de prospectos
- **Integré con sistemas de inventario** para evaluar capacidad logística de distribuidores
- **Diseñé dashboard en tiempo real** que muestra métricas de captación durante eventos
- **Optimicé performance para uso intensivo** durante horarios pico en ferias comerciales
