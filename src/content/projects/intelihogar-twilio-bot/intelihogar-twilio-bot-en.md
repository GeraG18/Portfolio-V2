# InteliHogar Twilio InteliBot

**InteliHogar Twilio InteliBot** is an automated emergency call system developed for Córdova Industrial Integradores' (CIDTcor) InteliHogar security ecosystem, designed to address the scenario of users without mobile data when leaving their homes. This innovative solution makes up to three automatic call attempts when an alarm is activated, offering users the option to deactivate the alarm through voice commands or authorize reporting to emergency services, ensuring effective response even without internet connection.

## Main Features

- Automated emergency call system
- Up to three contact attempts per activated alarm
- Voice command interaction for deactivation
- Direct integration with emergency services
- Continuous monitoring of locations with alarms

### Key Responsibilities and Achievements:

- **Developed the Firebase function** that continuously monitors locations and activates the Twilio system for alarms
- **Implemented complete Twilio automation** programming all call interaction functionalities
- **Conducted comprehensive system testing** verifying the correct functioning of the complete call sequence
- **Managed production deployment** implementing and monitoring the system in the final operational environment
