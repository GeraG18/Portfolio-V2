# Aplicación Cord

**Cord** es un proyecto innovador desarrollado por el área de Innovación y Desarrollo Tecnológico de Córdova Industrial (CIDTcor), cuyo objetivo es transformar la automatización empresarial mediante soluciones de Industria 4.0.

Utilizando tecnología IoT de vanguardia, Cord permite a las empresas optimizar sus operaciones de manufactura a través de una aplicación integral de gestión. La plataforma ofrece monitoreo en tiempo real de variables críticas de producción, control remoto de dispositivos conectados y un sistema de alertas personalizado disponible 24/7 para garantizar la continuidad operativa.

## Características principales

- Monitoreo de equipos PLC de zonas industriales.
- Alertas personalizadas sobre los parámetros clave de cada dispositivo.
- Control remoto de equipos desde cualquier parte del mundo.
- Seccionamiento de equipos por zonas para una mejor organización.
