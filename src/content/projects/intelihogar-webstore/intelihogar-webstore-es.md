# Tienda Web InteliHogar

**Tienda Web InteliHogar** es la plataforma de comercio electrónico desarrollada para el ecosistema de casas inteligentes de Córdova Industrial Integradores, diseñada para facilitar a los usuarios la adquisición completa de sistemas de automatización del hogar. La tienda online presenta el catálogo completo de productos disponibles para crear casas inteligentes, permite la contratación de suscripciones de mantenimiento y vigilancia 24/7, y gestiona todo el proceso de compra e instalación, ofreciendo una experiencia integrada desde la selección de productos hasta el soporte continuo.

## Características principales

- Catálogo completo de productos para casas inteligentes
- Sistema de suscripciones para mantenimiento y monitoreo 24/7
- Proceso de compra integrado con múltiples opciones de pago
- Gestión completa de pedidos y servicios contratados
- Diseño responsivo optimizado para escritorio y móvil

### Responsabilidades y Logros Clave:

- **Implementé el nuevo modelo de negocio** integrando pagos por suscripción y compra de dispositivos en una plataforma unificada
- **Integré el sistema de pagos Stripe** implementando todas las funcionalidades de procesamiento de transacciones y seguridad
- **Diseñé y desarrollé la interfaz de escritorio** creando una experiencia de compra optimizada para usuarios de computadora
- **Desarrollé algoritmos de cálculo** para sistemas de suscripción, incluyendo proyecciones de pagos mensuales y análisis financieros
