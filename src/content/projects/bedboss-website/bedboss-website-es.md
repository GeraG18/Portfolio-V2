# BedBoss Website

**BedBoss Website** es la plataforma de comercio electrónico desarrollada para BedBoss, fabricante especializado en camas y accesorios para camionetas dentro del grupo corporativo H7X. Este sitio está diseñado para el mercado de entusiastas de vehículos y propietarios de flotas, ofreciendo una experiencia de compra personalizada con configuradores visuales, compatibilidad por modelo de vehículo y herramientas para visualizar productos instalados.

## Características principales

- Configurador visual de camas por modelo de camioneta
- Sistema de verificación de compatibilidad automática
- Galería interactiva de instalaciones reales con filtros por modelo
- Sistema de e-commerce con opciones de financiamiento y arrendamiento
- Biblioteca de instalación y mantenimiento con tutoriales en video
- Integración con sistemas de gestión de flotas vehiculares

### Responsabilidades y Logros Clave:

- **Diseñé experiencia de compra visual única** con configurador 3D de camas para camionetas
- **Implementé sistema de compatibilidad inteligente** verificando automáticamente ajustes por modelo
- **Desarrollé galería interactiva** mostrando instalaciones reales filtradas por vehículo
- **Optimicé para alto tráfico de imágenes** implementando lazy loading y CDN especializado
- **Integré opciones de financiamiento flexible** adaptadas al mercado automotriz
- **Creé sistema de recomendaciones cruzadas** aumentando el valor promedio del carrito
