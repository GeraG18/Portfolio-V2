# Backend InteliHom

**Backend InteliHom** es la versión internacionalizada del sistema de administración desarrollado para la expansión al mercado estadounidense de la plataforma de casas inteligentes de Córdova Industrial Integradores. Este sistema, derivado de InteliHogar, proporciona todas las funcionalidades administrativas necesarias para gestionar localizaciones de usuarios, dispositivos conectados, productos de la tienda online y demás operaciones del ecosistema, específicamente adaptado para las necesidades y normativas del mercado estadounidense.

## Características principales

- Sistema completamente internacionalizado para inglés estadounidense
- Gestión administrativa adaptada al mercado internacional
- Todos los componentes traducidos y localizados
- Infraestructura preparada para expansión global
- Mantenimiento de funcionalidades del backend original

### Responsabilidades y Logros Clave:

- **Implementé el sistema de internacionalización (i18n)** en Angular configurando la infraestructura completa para soporte multilingüe
- **Realicé la traducción completa del backend** adaptando todos los textos, etiquetas y mensajes al inglés para el público estadounidense
- **Gestioné el despliegue en producción** implementando y configurando el sistema específicamente para el mercado internacional objetivo
