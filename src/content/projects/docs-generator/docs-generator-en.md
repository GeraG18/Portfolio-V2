# Calculation Report Generator Script

**Calculation Report Generator Script** was a tool developed for Córdova Industrial Integradores, designed to generate multiple technical documents from data extracted from Excel files (".xlsx" and ".csv"). The system performed complex calculations to create detailed calculation reports, subsequently converting them to Word formats (".doc" and ".docx") ready for delivery. Developed in JavaScript with Node.js, it used the "docx" library for document generation.

## Main Features

- Automated generation of technical documents from Excel files
- Conversion of complex calculations to structured Word documents
- Processing of multiple input formats (.xlsx, .csv)
- Export to compatible .doc and .docx formats

### Key Responsibilities and Achievements:

- **Designed and developed the document structure** in JavaScript, creating customized templates for calculation reports
- **Implemented automated script execution** verifying the correct generation of all documents
- **Optimized and corrected design issues** in the generated Word files, ensuring visual consistency
