# Script Generador de Memorias de Cálculo

**Script Generador de Memorias de Cálculo** fue una herramienta desarrollada para Córdova Industrial Integradores, diseñada para generar múltiples documentos técnicos a partir de datos extraídos de archivos Excel (".xlsx" y ".csv"). El sistema realizaba cálculos complejos para crear memorias de cálculo detalladas, convirtiéndolas posteriormente a formatos Word (".doc" y ".docx") listos para entrega. Desarrollado en JavaScript con Node.js, utilizó la librería "docx" para la generación de documentos.

## Características principales

- Generación automatizada de documentos técnicos desde archivos Excel
- Conversión de cálculos complejos a documentos Word estructurados
- Procesamiento de múltiples formatos de entrada (.xlsx, .csv)
- Exportación a formatos .doc y .docx compatibles

### Responsabilidades y Logros Clave:

- **Diseñé y desarrollé la estructura de documentos** en JavaScript, creando plantillas personalizadas para memorias de cálculo
- **Implementé la ejecución automatizada del script** verificando la correcta generación de todos los documentos
- **Optimicé y corregí problemas de diseño** en los archivos Word generados, asegurando la consistencia visual
