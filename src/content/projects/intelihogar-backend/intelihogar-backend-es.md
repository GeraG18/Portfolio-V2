# Backend InteliHogar

**Backend InteliHogar** es el sistema de administración desarrollado para la plataforma de casas inteligentes de Córdova Industrial Integradores, diseñado para gestionar todos los aspectos administrativos del ecosistema InteliHogar. Esta herramienta permite a los administradores crear y gestionar localizaciones de usuarios, administrar dispositivos conectados, agregar nuevos productos a la tienda online y controlar todas las funcionalidades operativas del sistema, proporcionando un control centralizado sobre toda la infraestructura de casas inteligentes.

## Características principales

- Gestión completa de localizaciones y usuarios
- Administración centralizada de dispositivos IoT
- Sistema de inventario y gestión de productos
- Control de existencias de la tienda online
- Automatización de configuraciones del sistema

### Responsabilidades y Logros Clave:

- **Desarrollé el sistema de vinculación de dispositivos** permitiendo la asociación automática de dispositivos IoT con localizaciones específicas para visualización en la aplicación
- **Implementé el sistema de creación de localizaciones** desde el backend con asignación automática de propietarios
- **Diseñé y agregué nuevos campos de productos** desarrollando un script para actualizar productos existentes con la nueva estructura de datos
- **Creé la lógica de gestión de inventario** implementando un sistema para identificar y marcar productos sin existencias en la tienda online
- **Desarrollé sistemas de automatización** para agregar funcionalidades como la alarma inteligente con configuración mediante un solo clic
