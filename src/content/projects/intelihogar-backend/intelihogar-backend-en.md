# InteliHogar Backend

**InteliHogar Backend** is the administration system developed for Córdova Industrial Integradores' (CIDTcor) smart home platform, designed to manage all administrative aspects of the InteliHogar ecosystem. This tool allows administrators to create and manage user locations, administer connected devices, add new products to the online store, and control all operational functionalities of the system, providing centralized control over the entire smart home infrastructure.

## Main Features

- Complete management of locations and users
- Centralized administration of IoT devices
- Inventory and product management system
- Online store stock control
- System configuration automation

### Key Responsibilities and Achievements:

- **Developed the device linking system** allowing automatic association of IoT devices with specific locations for application visualization
- **Implemented the location creation system** from the backend with automatic owner assignment
- **Designed and added new product fields** developing a script to update existing products with the new data structure
- **Created inventory management logic** implementing a system to identify and mark out-of-stock products in the online store
- **Developed automation systems** to add functionalities like the smart alarm with one-click configuration
