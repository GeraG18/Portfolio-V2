# Aplicación de Monitoreo InteliHogar

**Aplicación de Monitoreo InteliHogar** es una herramienta especializada desarrollada como extensión del sistema de alarmas inteligentes de Córdova Industrial Integradores, diseñada exclusivamente para el personal de monitoreo de seguridad. Esta aplicación de uso privado permite a los operadores identificar instantáneamente cuando una alarma se activa en cualquier hogar del sistema InteliHogar, proporcionando la información necesaria para tomar acciones preventivas inmediatas y proteger la integridad de los usuarios y sus propiedades.

## Características principales

- Monitoreo en tiempo real de activaciones de alarmas
- Visualización inmediata de localizaciones con alertas activas
- Acceso a datos completos de usuarios y propiedades
- Interfaz optimizada para respuesta rápida en emergencias
- Sistema privado restringido a personal autorizado

### Responsabilidades y Logros Clave:

- **Diseñé y desarrollé la interfaz de usuario completa** creando una experiencia intuitiva y eficiente para el personal de monitoreo
- **Implementé animaciones en CSS en tiempo real** para mostrar visualmente las localizaciones con alarmas activadas
- **Desarrollé el sistema de visualización de datos** presentando información crítica del propietario y propiedad para facilitar la creación de reportes de emergencia
