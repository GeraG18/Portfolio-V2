# InteliHogar Monitoring Application

**InteliHogar Monitoring Application** is a specialized tool developed as an extension of Córdova Industrial Integradores' (CIDTcor) smart alarm system, designed exclusively for security monitoring personnel. This private-use application allows operators to instantly identify when an alarm is activated in any InteliHogar system home, providing the necessary information to take immediate preventive actions and protect user integrity and properties.

## Main Features

- Real-time monitoring of alarm activations
- Immediate visualization of locations with active alerts
- Access to complete user and property data
- Interface optimized for rapid emergency response
- Private system restricted to authorized personnel

### Key Responsibilities and Achievements:

- **Designed and developed the complete user interface** creating an intuitive and efficient experience for monitoring personnel
- **Implemented real-time CSS animations** to visually display locations with activated alarms
- **Developed the data visualization system** presenting critical owner and property information to facilitate emergency report creation
