# Tienda Web InteliHom

**Tienda Web InteliHom** es la plataforma de comercio electrónico internacionalizada desarrollada para la expansión al mercado estadounidense del ecosistema de casas inteligentes de Córdova Industrial Integradores. Esta tienda online, adaptada específicamente para consumidores estadounidenses, presenta el catálogo completo de productos para crear casas inteligentes, permite la contratación de suscripciones de mantenimiento y vigilancia 24/7, y gestiona todo el proceso de compra con interfaces y contenidos completamente localizados para el público objetivo.

## Características principales

- Plataforma completamente internacionalizada en inglés
- Catálogo de productos adaptado al mercado estadounidense
- Sistema de suscripciones localizado para normas y regulaciones de EE.UU.
- Proceso de compra optimizado para consumidores internacionales
- Contenidos y marketing específicos para el mercado objetivo

### Responsabilidades y Logros Clave:

- **Implementé el sistema de internacionalización (i18n)** en Angular configurando toda la infraestructura para soporte multilingüe
- **Realicé la traducción completa de la plataforma** adaptando todos los textos, interfaces y contenidos al inglés para el público estadounidense
- **Gestioné el despliegue en producción** implementando y configurando la tienda específicamente para el mercado internacional objetivo
